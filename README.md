# deepseek 3.2 落地页

新一代开源大语言模型 deepseek 3.2 的官方落地页，展示其在推理能力、成本效益和技术创新方面的核心优势。

- **推理能力对标GPT-5**: 标准版达到GPT-5级别，Speciale增强版媲美Gemini 3.0 Pro
- **DSA稀疏注意力机制**: 长文本处理成本降低60%，推理速度提升3.5倍
- **128K超长上下文**: 支持大规模文档分析和复杂任务处理
- **深度思考模式**: 展示完整推理过程，答案更严谨可靠
- **完全开源**: 支持私有化部署，数据安全可控
- **国际竞赛金牌**: 在IMO、ICPC等顶级竞赛中获得金牌

## 🚀 部署说明

### 自动部署（推荐）
1. Fork本项目到你的GitHub账户
2. 在Vercel中导入项目
3. 设置自动部署触发

### 手动部署
1. 克隆项目到本地
2. 上传文件到Web服务器
3. 配置服务器端HTTP头部（参考`.htaccess`或`nginx-config.conf`）

### 重要说明
- **X-Robots-Tag**: 需要在Web服务器层面配置，详见`README-XRobotsTag.md`
- **PWA功能**: 确保HTTPS协议正常工作
- **域名配置**: 修改所有硬编码的URL为实际域名
